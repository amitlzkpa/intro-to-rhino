# Topics
```
# ABOUT RHINO
- NURBS vs. other CAD software

## Interface
- Command line
- Viewports, view modes
- Layers and Properties
- Preferences (units, grid, etc.)
- Snaps (O-snap, grid snap, etc.)
- Gumball
- Enter or Right click to repeat command


## RHINO USES
- Modeling for rendering
- Technical modeling for analysis
- Data visualization
- Platform for parametric modeling with Grasshopper




# GEOMETRY - 2D
## Draw
- Point
- Line
- Polyline
- Polygons
- NURBS curve
   - About
   - Hatch to fill


## Curve Editing
- Trim/Extend
- Fillet/Join
- Offset
- Intersect
- Divide/Split
- Control points
- Rebuild
- Mirror
- Explode (polyline)
- Group


## Curve Analysis
- Dims/distance
- Area


## Transformations
- Move/copy
- Rotate (in-plane)
- Array (rectangular, polar)
- Scale 1D/2D (Gumball shortcut)
```
